# Gendiff

## Description
A CLI utility that compares two configuration files and shows a difference.

## Installation
```bash
git clone https://github.com/nnsquik/java-project-71.git
cd java-project-71
./gradlew installDist
```

# Make commands
```bash
make run-dist # запуск приложения
make diff     # запуск с тестовыми файлами
```
